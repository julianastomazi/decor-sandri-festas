# Decor Sandri

Site institucional da Decor Sandri, especializada em decorações de festas infantis e adultas em São Leopoldo e região.

## Desenvolvimento local

```bash
pnpm install
pnpm dev
```

## Publicação

O site é exportado de forma estática e publicado automaticamente no GitHub Pages pela rotina configurada em `.github/workflows/pages.yml`.
